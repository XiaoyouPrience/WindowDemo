
# WindowDemo

#这是一个类似于网易邮箱大师一开起就有一个密码框一样的效果，今天想了一下封装了一个简单的效果。

#使用很方便，值需要在 APPDelegate.m 中导入一个XYWindow.h 然后在- (void)applicationDidBecomeActive:(UIApplication *)application {} 
#方法中添加一行代码 [[XYTempWindow window] show];即可。
